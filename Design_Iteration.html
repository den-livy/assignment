Iteration 1: Optimizing Accordion Fold Count
VERSION 1 - REJECTED
[DRAW: Bottle with 3 shallow folds]
3 folds, each ~0.8" deep
  • ✗ Only 30% height reduction
  • ✗ Doesn't collapse enough
  • ✗ Still too bulky when empty
VERSION 2 - REJECTED
[DRAW: Bottle with 6 thin folds]
6 folds, each ~0.5" deep
  • ✓ Excellent compression (70%)
  • ✗ Too flexible/unstable when full
  • ✗ Collapses under water weight
VERSION 3 - FINAL ✓
[DRAW: Bottle with 4 medium folds]
4 folds, each ~0.9" deep
  • ✓ 60% height reduction
  • ✓ Stable structure when filled
  • ✓ Perfect balance achieved
Testing Method: Created paper mockups at 1:1 scale, filled with water to test stability. Version 3 (4 folds) provided optimal balance between compactness and structural integrity. Folds deep enough for meaningful size reduction but not so numerous that bottle becomes floppy.
